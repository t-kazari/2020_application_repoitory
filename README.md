# 2020_application_repoitory
## ○ 概要
食事提案をしてくれるデスクトップアプリです。
名付けて、「Food Composer」
元々は「Food Proposer」だったが、スペルミス（というか単語ミス）をしていることに気づかずいろいろ進めちゃったし、~~なんか面白いから~~もうそのままいこうということに。

モバイル端末からの利用とか知ったこっちゃないです。あと、極力GCPの無料枠だけでやるつもりです。
